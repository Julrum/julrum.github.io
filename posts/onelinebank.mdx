2021 우리은행 온(On)택트 해커톤 본선대회 출품작

## 프로젝트 소개

은행 어플을 통한 서비스가 많아지면서 UI가 복잡해지고 있고 그로 인해 사용성이 많이 떨어집니다.
이에 송금 서비스를 간편하게 사용할 수 없을까 생각을 하였고 우리에게 익숙한 채팅 방식을 도입하였습니다.
보낼 사람의 은행과 계좌번호, 금액을 입력한 후 확인 과정과 인증 과정을 거쳐 송금을 보낼 수 있습니다.
또한 주소록에 자주 보내는 사람의 은행명과 계좌번호를 입력후 채팅에서 해당 사람의 이름만 입력하면 바로 송금할 수 있도록 구현하였습니다.

### 사용한 기술 스택

React Native, Google Firebase

### 담당 역할

- **Main**
  - OpenWeatherMap API에서 통신하여 가져온 Data를 RecyclerView에 연결하였습니다.
  - 사용자의 위치에 기반한 날씨를 제공하기 위해 SharedPreference에 사용자 위치 정보(위도, 경도)를 저장하였습니다.
  - 다른 Activity로 Intent를 전달하고 연결하며 전체 Flow를 연결하였습니다.
- **Settings**
  - LocationManager를 이용해 Main에서 필요한 사용자의 위치 정보를 현재 위치로 설정하였습니다.
  - VWorld 검색 API와 SearchView를 이용하여 주소 검색 기능을 구현하였습니다.
- **Splash**
  - 앱 실행 시 정보 오류에 의한 Runtime Error가 발생하지 않도록 인터넷 연결 및 위치 접근 권한을 확인하였습니다.
